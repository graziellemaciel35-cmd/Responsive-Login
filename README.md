# 🔐 Responsive Login Form

Formulário de login moderno, responsivo e com validação de campos, desenvolvido com HTML e CSS.

---

## ✨ Funcionalidades
- Layout responsivo (funciona em celulares, tablets e desktops)
- Validação de campos obrigatórios com mensagem de alerta
- Design limpo e intuitivo
- Campos para e-mail/telefone e senha
- Links para "Esqueceu a senha?" e cadastro de novo usuário
- Estilização com cores e ícones visuais

---

## 🗂️ Estrutura de Arquivos
 
 
responsive-login/
├── index.html      # Estrutura do formulário
├── style.css       # Estilos, cores e responsividade
└── script.js       # Validação básica dos campos
 
plaintext
  

---

## 🚀 Como usar
1. Baixe ou clone este repositório
2. Abra o arquivo index.html diretamente no navegador
3. Para testar a validação: clique no botão "Login" sem preencher os campos — será exibida a mensagem de alerta
4. Personalize cores, textos e tamanhos editando o arquivo style.css

---

## 🛠️ Tecnologias utilizadas
- *HTML5*: Estrutura semântica do formulário
- *CSS3*: Estilização, responsividade e layout
- *JavaScript*: Validação básica de campos obrigatórios

---

## 🎨 Personalização
- *Cores*: Altere as variáveis de cor no início do arquivo style.css
- *Textos*: Modifique diretamente no arquivo index.html
- *Validação*: Adicione regras mais avançadas no arquivo script.js (ex: formato de e-mail, força da senha)

---

## 📌 Observação
Este é um projeto de interface (front-end). Para um sistema completo, é necessário integrar com um back-end para verificar credenciais e gerenciar sessões de forma segura.
